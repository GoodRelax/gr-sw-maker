# create-gr-sw-maker

Scaffold a [gr-sw-maker](https://github.com/GoodRelax/gr-sw-maker) project.

## Usage

```bash
npm init gr-sw-maker my-project
cd my-project
bash setup.sh ja    # or: en
```

Then write your concept in `user-order.md` and run `/full-auto-dev`.

## What it does

1. Downloads the latest gr-sw-maker template from GitHub
2. Extracts it into the specified directory
3. That's it — no dependencies installed, no config generated
